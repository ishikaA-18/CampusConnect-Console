Campus Connect — Console Edition
A console-based system that helps students locate teachers and navigate campus easily.
Why I Built This
In my third semester I saw a student knocking on our ongoing class looking for a teacher. Our lecturer got annoyed. I wanted to build something that would prevent that kind of confusion — especially for freshers who don't know the campus yet.
Features So Far

Faculty directory with real teacher data
Search teachers by department
Displays teacher name, room, designation and subject
